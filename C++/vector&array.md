# vector
## 선언 및 초기화
> `vector<int> v;`
> 
> `vector<int> v(5);` : 5개 원소
> 
> `vector<int> v(5, 2);` : 2로 초기화 된 5개 원소
> 
> `vector<int> v(3) = {1, 2, 3};` : 원소 초기화
> 
> `vector<int> v = {1, 2, 3};` : 원소 초기화
> 
> `vector<int> v2(v1);` : 복사 생성자

## 원소 할당/복사
> `v.assign(5, 2);` : 5개 원소를 2로
> 
> `v[0] = 3` : 0번째 원소를 3으로
> 
> `v2.assign(v1.begin(), v1.end())` : 전체 복사
> 
> `v2.assign(v1.begin()+1, v1.begin()+5)` : 0, __1, 2, 3, 4,__ 5 번째 중에서 1, 2, 3, 4 번째 원소 복사

# array
## 선언 및 초기화
> `int a[5];` : 5개의 칸(불변)을 가진 array
> 
> `int a[5] = {1, 2, 3};` : 1, 2, 3, 0, 0 으로 원소 초기화
> 
> `int a[] = {1, 2, 3};` : 원소 초기화
> 
> 복사 생성자 없음. => `memset` 사용

## 원소 할당/복사
> `a[0] = 3;` : 0번째 원소를 3으로
> 
> `*(a+2) = 3` : 2번째 원소를 3으로
> 
> `memset(a, 3, sizeof(a));` : a의 모든 원소를 3으로 변경
> 
> `memcpy(dst, src, sizeof(src));` : src를 dst에 복사
